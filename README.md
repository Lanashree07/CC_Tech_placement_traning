# CC Tech Placement Training

This repository contains C programming practice programs completed as part of my **CC Tech Placement Training**.
It includes basic to intermediate C programs for improving programming logic and preparing for technical interviews and campus placements.

## 📚 Language

- C Programming

## 📂 Topics Covered

- Variables and Data Types
- Input and Output
- Operators
- Conditional Statements
- Looping Statements (`for`, `while`, `do-while`)
- Pattern Programs
- Number Programs

## 🚀 How to Use

1. Clone this repository.
2. Open the source files in VS Code or any C IDE.
3. Compile and run the programs using a C compiler such as GCC or MinGW.

## 🎯 Purpose

- Strengthen C programming fundamentals.
- Improve logical thinking and problem-solving skills.
- Prepare for coding interviews and placement assessments.
